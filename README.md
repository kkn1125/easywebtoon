# EasyWebtoon

이지웹툰은 과거에 인기가 있던 이지툰이라는 로토스코핑 프로그램을 모방한다.

웹에서 그림을 그리고, 프레임마다 가이드라인을 따라 간단한 프레임 애니메이션을 구현할 수 있다.

단순한 조작과 어렵지 않은 구성으로 모두가 편리하게 사용하는 것을 목표로 한다.

## 개선사항

추후 데이터베이스에 애니메이션을 저장하고, 서로 공유하는 서비스를 추가할 예정이다.
